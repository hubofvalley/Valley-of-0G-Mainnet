# Storage KV Guide

Deploy and manage a 0G Storage KV (Key-Value) service node.

## Overview

Storage KV nodes provide key-value storage services, maintaining KV streams for applications built on 0G.

## System Requirements

| Category | Requirements |
|----------|--------------|
| CPU | 8+ cores |
| RAM | 32+ GB |
| Storage | Matches the size of KV streams maintained |

## Installation

1. Launch Valley of 0G:
   ```bash
   bash <(curl -s https://raw.githubusercontent.com/hubofvalley/Mainnet-Guides/main/0g%20\(zero-gravity\)/resources/valleyof0G.sh)
   ```
2. Select **"Storage KV"** → **"Deploy Storage KV"**
3. Follow the interactive prompts

**Current Version**: v1.4.0

## Updating

1. Launch Valley of 0G
2. Select **"Storage KV"** → **"Update Storage KV"**
